// Getting started checklist (VSCode + GitHub + Deploy)

### Initialize project
```bash
npx create-react-app fender-play-ui --template typescript
cd fender-play-ui
npx storybook@latest init
```

### Install tooling
```bash
npm install --save-dev prettier eslint stylelint vite vite-tsconfig-paths
npm install --save @storybook/addon-essentials @storybook/addon-interactions
```

### Launch Storybook
```bash
npm run storybook
```

### Figma
- Create a Figma project: "Fender Play Design System"
- Use styles for tokens (color, text)
- Use components for each UI element
- Link Figma files in Storybook pages

### Deploy options
- Chromatic for visual hosting and testing
- `build-storybook` + Vercel/Netlify/GitHub Pages

### GitHub Actions (deploy-storybook.yml)
- Build and deploy `storybook-static/` folder on push
